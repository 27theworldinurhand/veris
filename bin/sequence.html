Change Sequence of VERIS 2.0 Enumerations


VERIS version 2 adds the ability to sequence many of the enumerations in the veris schema. Old records may not have this sequencing, requiring they be manually updated. To use:
  1. Choose the input and output directories, (which currently must be different), and click 'load'. This will populate the file list with only those files not in both directories.
  2. Select a file in the files list. This will build a sequence table.
  3. Use the radio buttons to choose each enumeration should be. (Each sequence step must have either an action or an attribute.)
  4. Click 'save' to write the file to the output and remove it from the list of files.
